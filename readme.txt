

ScrabbleAR

Juego desarrollado en la materia Seminario de Lenguajes . Está basado en el
ya conocido juego “Scrabble” pero cuenta con sus propias reglas, detalles y, en
particular, esta es una implementación íntegramente hecha en python. Respetando
los principios del software libre, pudiendo este ser  ser estudiado, modificado, y
utilizado libremente con cualquier fin y redistribuido con cambios o mejoras o sin ellas. 
	



Comenzando 🚀


Pre-requisitos 📋

Que cosas necesitas para instalar el software y como instalarlas
		Python 3.6 – se puede instalar desde su pagina web https://www.python.org/
		PysimpleGui 3.5 o mayor  – hay diversas formas de instalarlo, la que nos parece mas simple es haciendolo con pip . Para esto vamos desde el simbolo del sistema hasta la carpeta donde se encuentra descargado pip ( se descarga con el paquete de Python), y ejecutamos 
	pip install pip
y luego ejecutamos la siguiente linea
	pip install PySimpleGUI
